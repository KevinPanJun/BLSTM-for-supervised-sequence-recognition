Input Feature size vs Space occupied by network

The space required by the network is the same as the space required to store the weight of each edge

Since total number of weights is W = 4IH + 4H2 + 3H + HK we expect a linear dependancy between Number of input features and Space

As the image shows, the relationship is linear with a small positive slope

In the range of 10 to 3000, the size varies from 29 to 270 Mb

For this experiment we use a network with 100 Hidden cells and 50 Output blocks
