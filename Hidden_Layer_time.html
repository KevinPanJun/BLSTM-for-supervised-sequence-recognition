Hidden Layer size vs Time for training network

The time complexity of training one pair of input and output sequences is of the order of the number of Weights, since we have to evaluate the weighted sum in forward pass and update all weights after backward pass

Since total number of weights is W = 4IH + 4H2 + 3H + HK we expect a quadratic dependancy between Number of Hidden Cells and Time

For this experiment we use a data set of consisting of 20 sequences

As the image shows, the relationship quadratic

In the range of 10 to 3000, the size varies from 0.2 to 70 seconds

For this experiment we use a network with 15 Input Features and 173 Output classes
