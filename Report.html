Study of BLSTM for Sequence Labelling

The implementation of BLSTM in OCRUPUS can be modified and extended so as to use it for recognizing any language.

First this report presents a few experiments which show the space and time dependencies of BLSTM neural network and also a study of network performance in case of extracted feature input compared to raw feature input.

Then a simple interface for training and testing a BLSTM neural network is presented at the ending.

Features

using this modified package, we can do the following things:

  • Train a BLSTM network for a fixed number of epochs
  • Train a BLSTM network until the label error rate is less than a specified threshold error on a validation input
  • Retrain previosly saved network using any of the above two methods
  • Test a saved network on testing data
  • Get detailed reports on input-output and error rates in each epoch
  • Save both network with least error and network after last epoch
  • lstm.py is much more generic and contains Logreg,Softmax,MLP and LSTM neural network implementatons. We can also build networks of any size and shape using Stacked, Parallel and Reverse interfaces provided in lstm.py
