Input Feature size vs Time for training network

The time complexity of training one pair of input and output sequences is of the order of the number of Weights, since we have to evaluate the weighted sum in forward pass and update all weights after backward pass

Since total number of weights is W = 4IH + 4H2 + 3H + HK we expect a linear dependancy between Number of input features and Time

For this experiment we present the same randomly generated input sequence and output sequence of 10 timesteps 300 times

As the image shows, the relationship is linear with a small positive slope

In the range of 10 to 3000, the size varies from 2 to 24 seconds

For this experiment we use a network with 100 Hidden cells and 50 Output blocks
